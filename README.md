# CS1804T – Data Structures Lab  
Shiv Nadar University, Chennai

This repository contains my lab solutions for the course **CS1804T – Data Structures Lab**.  
All programs are written in **C++** and follow the lab instructions given during the semester.
  
---  

## Contents  
### Lab 1 - Searching and Sorting
1. Sorting algorithms (Bubble, Selection, Insertion)
2. Search algorithms (Linear and Binary Search)
### Lab 2  - Classes and Pointers Recap  
Dynamic memory allocation exercises and class-based programs for 
1. Number properties (Palindrome, Armstrong, Perfect)
2. Geometric shape calculations (Square, Cube, Rectangle, Cuboid)
### Lab 3 - List ADT - Array Implementation  
Array-based List ADT with insert, delete, search, display, and rotate operations. Includes multiple solutions for array rotation with O(1) space complexity  
### Lab 4 - List ADT - Singly Linked List  
Singly linked list implementation with basic operations, reverse printing, link reversal, and sorted list merging functionality  
### Lab 5 - List ADT - Circular and Doubly Linked List
1. Implementation of doubly linked lists with tail pointer
2. Implementation of circular linked lists
3. Round-robin scheduling simulator using circular lists  
### Lab 6 - Stack ADT - Array and Linked List
1. Stack implementation using arrays
2. Stack implementation using linked lists
3. Program for infix to postfix conversion, postfix evaluation, and parenthesis balance checking  
### Lab 7 - Queue ADT - Array and List Implementation
1. Queue implementation using arrays  
2. Circular queue implementation using arrays
3. Linked list-based queue
4. Symbol processing program
### Lab 8 - Tree ADT - Binary Tree  
1. [Block Towers Problem - Codeforces](https://codeforces.com/problemset/problem/1767/B)  
2. Binary tree implementation with traversal methods (Preorder, Inorder, Postorder) and search functionality  
### Lab 9 - Tree ADT - Binary Search Tree  
1. [Nene's Game - Codeforces](https://codeforces.com/problemset/problem/1956/A)  
2. [Advantage - Codeforces](https://codeforces.com/problemset/problem/1760/C)
3. Binary Search Tree implementation with traversals and search operations
4. Expression tree construction from postfix expressions
### Lab 10 - Priority Queue ADT - Heap
1. [Remove Prefix - Codeforces](https://codeforces.com/problemset/problem/1714/B)
2. [Word Game - Codeforces](https://codeforces.com/problemset/problem/1722/C)
3. Implementation of Priority Queue ADT using a max heap  
### Lab 11 - Hash ADT - Hash Table
Hash table implementations using three collision resolution techniques: Linear Probing, Quadratic Probing, and Separate Chaining  
### Lab 12 - Graph ADT - Adjacency List and Matrix  
1. Graph implementation using adjacency matrix  
2. Graph implementation using adjacency list  
3. Implementations of Prim's, Kruskal's, and Dijkstra's algorithms
